Aide-mémoire JavaScript – Module 2B
Fonctions avancées

1

Le mot-clé this

Dans un objet :

const user = {
  name: "Habib",
  greet() {
    console.log("Bonjour " + this.name);
  }
};

Arrow functions et this

Les arrow functions n'ont pas de this personnel.

const obj = {
  x: 10,
  show: () => this.x   // undefined
};
2

Valeurs par défaut

function multiply(a, b = 1) {
  return a * b;
}
3

Paramètres rest

function sum(...nums) {
  return nums.reduce((acc, n) => acc + n, 0);
}
4

Hoisting

Hoisted :

hello(); // OK
function hello() {}

Non hoisted correctement :

hello(); // Erreur
const hello = function() {};
5

Fonctions vs Méthodes

function f() {}        // fonction
const obj = { f() {} } // méthode
