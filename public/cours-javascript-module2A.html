Aide-mémoire JavaScript – Module 2A
Les Fonctions (bases)

1

Pourquoi les fonctions ?

  • Réutiliser du code.
  • Organiser la logique (propreté du code).
  • Réagir aux événements (clics, formulaires).
  • Manipuler des données (API, tableaux).
2

Les trois syntaxes de fonction

1) Déclaration classique

function hello() {
  console.log("Salut !");
}

2) Fonction expression

const hello = function() {
  console.log("Salut !");
};

3) Fonction fléchée

const hello = () => {
  console.log("Salut !");
};

Résumé :
Déclaration = hoisting.
Expression = plus moderne.
Arrow = concise, idéale pour callbacks.

3

Paramètres et arguments

function greet(name) {   // paramètre
  console.log("Bonjour " + name);
}

greet("Habib");          // argument
4

return (bases)

Retourne une valeur et stoppe l'exécution de la fonction.

function double(n) {
  return n * 2;
}

Fonctions dans le web

Validation

function isEmail(email) {
  return email.includes("@");
}

  console.log("Clic !");
