Aide-mémoire JavaScript – Module 2C
Le mot-clé return

1

Retour explicite

function double(n) {
  return n * 2;
}
2

Retour implicite (arrow)

Fonctionne :

const add = (a, b) => a + b;

Ne fonctionne pas :

const add = (a, b) => { a + b };  // undefined
3

Retour d’un objet (cas spécial)

Correct :

const getUser = () => ({ name: "Habib" });

Incorrect :

const getUser = () => { name: "Habib" }; // bloc vide → undefined
4

return dans les callbacks

map → retourne une nouvelle valeur

users.map(u => u.name);

filter → retourne true/false

users.filter(u => u.age >= 18);

find → retourne un élément

users.find(u => u.id === 1);

sort → retourne comparaison

numbers.sort((a, b) => a - b);
5

return dans une fonction async

Retourne toujours une Promise.

async function getUser() {
  return { name: "Habib" }; // → Promise.resolve(...)
}
6

return dans les closures

function createCounter() {
  let count = 0;
  return function() {
    count++;
    return count;
  };
}
